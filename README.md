# Mood-Chat-Responder-Reflex-Agent-
Small Python chatbot that can detect your mood based on what you type and reply in a friendly way. It uses a simple reflex agent approach — no memory, no learning — just quick mood detection from your words!

Features:
- Detects if your mood is happy, sad, or neutral
- Handles negations like “not happy” or “don’t hate” correctly
- Simple Tkinter GUI
- Scrollable chat window and text input box

Tech Stcak :
- Python
- Tkinter (for GUI)
